Prototype · NearShop Maps

Find shops on the map, see live distance, and tap one to start navigation.

This prototype shows the customer experience we can ship inside the app: a map with pins, distance badges, route steps, delivery status, and a bottom sheet that highlights the selected shop. The UX uses the saved preferred radius by default, so if the user never changes it, the app stays at 5 km.

Route
This prototype is intentionally close to the final in-app layout we can build with react-native-maps: a map view at the top, a selected shop bottom sheet, nearby shop cards below, live distance labels, route actions, and category/filter chips. The map itself can use either Google Maps or Mapbox.
If you want, the next step is to convert this into a real mobile screen and wire the markers to the existing nearby shops API, so the prototype becomes an actual working Maps tab.
